# Gravity Shift

**Gravity Shift** is a First-person shooter, multiplayer game where players will have to fight while experiencing gravity shifts.

---

## Features:

* **Multiplayer Networking** - UDP-based client-server architecture for real-time multiplayer
    - UDP Protocol for low-latency, fast-paced gameplay
    - Multiple clients can connect to a single server simultaneously
    - Server broadcasts messages to all connected players
* **Network Synchronization** - JSON serialization for reliable data transmission
    - Player positions
* **Real-Time Player Tracking** - Position updates sent over UDP with minimal latency
* **Scalable Architecture** - Extensible client-server framework for future features
